# VallCar Auto Vidros

Site institucional desenvolvido com Next.js e Tailwind CSS.

## Tecnologias

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Funcionalidades

- Home institucional
- Página de serviços
- Página de lojas
- Busca de unidades
- Google Maps
- WhatsApp integrado
- Formulário de orçamento
- Layout responsivo

## Como executar

```bash
npm install
npm run dev
```
